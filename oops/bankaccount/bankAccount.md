# Problem Description

Create a class named BankAccount that represents a customer’s bank account.
Each bank account should store basic details and allow common banking operations.

---

**Class Requirements:**

- Class Name: BankAccount

---

**Data Members (Attributes)**

	•	accountNumber – unique number of the account
	•	accountHolderName – name of the account holder
	•	balance – current balance in the account

**Methods (Behavior)**

	1.	deposit(amount)
        •	Adds the given amount to the balance.
        •	Display a success message after deposit.
	2.	withdraw(amount)
        •	Deducts the amount from the balance if sufficient funds are available.
        •	If balance is insufficient, display an appropriate message.
	3.	checkBalance()
    	•	Displays the current balance.
	4.	displayAccountDetails()
	    •	Displays account number, account holder name, and balance.

**Program Requirements**

	•	Create at least two objects of the BankAccount class.
    •	Assign initial values at the time of object creation. 
	•	Perform deposit and withdrawal operations on each account.
	•	Display updated account details after every operation.

**Sample Scenario:**

	•	Create an account for Anil with an initial balance of 10000.
	•	Deposit 2000 into the account.
	•	Withdraw 5000 from the account.
	•	Display the final account details.

**Expected Learning Outcomes:**

	•	Understanding class and object
	•	Encapsulation of data and behavior
	•	Using methods to manipulate object state
	•	Real-world modeling using OOP
